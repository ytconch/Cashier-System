# Cashier System
清水高中園遊會收銀系統

🛒 武告喝甲 - 攤位收銀與訂單管理系統

這是一個專為校慶攤位開發的 Web 自動化收銀系統，旨在解決尖峰時段的人力處理壓力，並即時追蹤涼麵與糖葫蘆的庫存與訂單狀態。

🌟 核心功能

即時訂單追蹤：透過 Web 介面同步更新所有訂單狀態（待處理、已完成）。

多裝置同步：支援多台行動裝置同時操作，收銀與出餐區資訊同步。

數據統計：自動加總銷售總額，方便活動結束後進行帳務核對。

🛠 技術架構

Frontend: HTML5, CSS3, JavaScript

Backend: Node.js

Database: SQLite / JSON Storage (視你的實作而定)

Authentication: JWT 安全驗證
